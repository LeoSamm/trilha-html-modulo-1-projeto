História dos computadores

Para dar inicio a história dos computadores é importante saber o que significa "computador", que seria algo que vem do verbo computar, então pensando assim podemos dizer que os computadores são maquinas de computar ou calcular dados. Sabendo disso podemos dizes que os computadores ja existem des das maquinas de calculos, como a Maquina de Babbage e a Maquina Turing.

Marcando a primeira geração dos computadores, em 1944 foi criado pelos Estados Unidos o Harvard mark 1 que foi considerado o primeiro computador, ele era baseado em valvulas o tornando eletro mecanico, e foi criado para realizar calculos bem mais complexos na época. Apos isso vieram o Colossus e o ENIAC, que ja eram um pouco mais automaticos. Os computadores dessa época eram gigantes ocupando salas inteiras, e foi nesta época surgiu o termo "bug", pois pelo fato desses computadores esquentarem muito, insetos eram atraidos e pousavam neles, e acabavam quebrando o equipamento.

A segunda geração dos computadores (1959 a 1965) foi marcada pelos Transistores que subistituiram as valvulas, fazendo com que os computares diminuisem de tamanho, podendo ser mechido entre salas, os tornando mais "portateis".

Ja na terceira geração (1965 a 1970) foram criados os Circuitos integrados que é composto por Transistores, Capacitores e Resistores, os famosos chips que fizeram os computadores diminuisem mais ainda e os tornando mais eficazes.

E por fim a quarta geração que se da de 1970 ate os dias atuais, nela teve a continuação a evoluçãos dos chips, a invenção do primeiro mouse desenvolvido pela apple, esta geração e muito marcada pelos Personal Computers/PCs que são os computadores mais pessoais, tornando os computadores mais populares e acessiveis.

O que são Clients

Falando da vida real, o cliente é a pessoa que tem acesso ou consome um certo produto ou serviço de uma empresa, falando-se de web, o Client é o dispositivo que consome alguma informação na internet, como por exemplo um tablet, um notebook, um smartphone ou um computador que se conectam por meio da internet a um site, que esta sendo hospedado em um servidor.

(Voltar)

O que são Servers

Quando falamos sobre servidores, podemos ter uma noção do que eles são apenas pelo seu nome, "servidor" nada mais é do que algo que serve informações para os Clients, esses servidores são nossos Servers, que são computadores voltados a isso.

Existem varios tipos de servidores, cada um com uma função, como por exemplo:

  • Servidores de Arquivos
  • Servidores de Seguranção(Firewall)
  • servidores de Streaming
  • Servidores de E-mail
  • Servidores Web
Um local cheio de servidores, é conhecido como Data Center.

Como é uma estrutura básica de um HTML

O HTML consiste em basicamente em três partes que são:

  1. A tag "< html >", que serve para dizer que é um codigo HTML
  2. A tag "< head >", tudo contido nesta tag, ira carregar antes de exibir pro usuario
  3. A tag "< body >", tudo contido nesta tag, é o que o usuario visualizara
Todas essas tags são abertas com "maior que, a tag, menor que" e apos abertas devem ser fechadas com "maior que, barra, a tag, menor que".

O que são as Tags e Atributos

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

Textos ou tipografía HTML

Existem varios HTML feitos exclusivamente para formatação de texto, basicamente eles sâo tags feitas para formatação, como por exemplo: "h1" até "h6", "p", "mark", "small", "i", "u", "strong", "sub", "sup", "blockquote"

Entendendo a diferença entre listas ordenadas e não ordenadas

No HTML existem dois tipos de listas, as listas ordenadas que fazem topicos numerados e as listas nao ordenadas que fazem topicos não numerados, suas tags são respectivamente "ol" e "ul", que devem ser abertas e fachadas ao terminar da lista, para fazer um novo topico é necessario usar a tag "li".
